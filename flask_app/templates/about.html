About:

This project serves as a member portal for any sort of organization that tries to keep members participating through the use of a points system. Largely inspired by my fraternity (hence the name), it includes not only point tracking, but also event creation onto a Google Calendar, the creation and completion of custom forms, basic account management functions such as profile pictures, and a member info lookup page.
One small note is that a user must be manually elevated to Admin role in the database, as this is the only secure way I could think about doing so.
